Archetypes

You need archetypes and optionally templates. Where you can get them:

  • You can get test archetypes from the openEHR knowledge2 Subversion repository , which includes:
    • ADL 1.5 validation archetypes and templates
    • openEHR example archetypes and templates
    • some early CEN EN13606 example archetypes
  • The openEHR archetypes are available from the Clinical Knowledge Manager (CKM). This repository has over 250 archetypes, in various stages of development by a large number of international experts, and includes:
    • clinical archetypes
    • administrative archetypes
    • demographic archetypes

See here for detailed set up instructions. Archetypes and templates can be put anywhere on your file system. If you are working on some archetypes or templates of your own that are based on existing archetypes, it is useful to put the latter in a 'reference' location, and your own in a 'work' location. You can configure ADL Workbench to know about reference and work locations, and it will visually indicate which archetypes are yours, and which are the reference ones.
