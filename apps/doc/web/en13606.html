Overview

An early version of an EN13606-1 schema for the AWB has been developed, along with some initial archetypes. This is intended to show how ADL / AOM 1.5 functions for archetypes built on a different reference model, and also to enable organisations working with EN13606 to have a more formal tool basis for developing archetypes, templates and generally migrating to ADL/AOM 1.5.

EN13606 schema for AWB

An experimental schema for EN13606-1 has been included in the set of RM schemas used by AWB. This was contributed by Christoph Rinner at the Medical University of Vienna . The definitive copy of this resides in the openEHR knowledge2 repository. If you are interested in being added to the committers list to work on this schema, please email .
